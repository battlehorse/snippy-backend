The best way to add your content and snippets to SnipBin is to install the Snippy extension for Google Chrome.

Install Snippy Chrome Extension

(you must use Google Chrome or Chromium for the extension to install successfully)

Usage

Once installed, you will see an extra marker icon in the toolbar. Click it to start grabbing snippets from any web page you visit.
Snippets can be anything! Product reviews from Amazon, excerpts from Wikipedia, citations to papers you care about, blog posts and much more.

Once you are satisfied with the snippets you have collected, click the marker icon again to review and edit them.

Once you are satisfied with your selection, upload it to SnipBin, and, if you want to, share it with the rest of the world! It's that easy!
